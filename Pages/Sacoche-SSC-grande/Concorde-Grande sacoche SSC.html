La grande sacoche de couleur écrue pour les Concorde




qui contient :
-  les enveloppes de couleur marron pour le Concorde
dans lesquelles seront rangés les dossiers de vol de l’étape précédente
-  les enveloppes rouges, même couleur pour tous les avions,
dans lesquelles seront rangés les rapports et compte-rendus
des étapes précédentes, dont le carton décollage/atterrissage remplis.
Ils sont archivés 6 mois avant destruction.
-  les cartons météo vierges qui seront utilisés par l’équipage
-  les cartons de décollages vierges
-  les suivis de vol vierges
-  le livret de consignes de sécurité en braille pour les passagers aveugles,

-  le livret d’avertissement pour les passagers indisciplinés,
-  le livret de gestion équipage, etc…
